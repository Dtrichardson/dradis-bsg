# DRADIS // Battlestar Galactica

A working DRADIS scope inspired by Battlestar Galactica.

- Rotating sweep with cone gradient
- Contacts: friendlies (▲ green), hostiles (◆ red), civilians (■ amber), unknown (○)
- Click contacts on the scope to read bearing / range / velocity
- Spawn contacts of any type
- ⚡ FTL JUMP reseeds the field
- WebAudio pings (different pitch per contact type)
- CRT scanline overlay

Stack: Next.js 16 + React 19 + Tailwind 4, deployed on Vercel.

```bash
npm install
npm run dev
```
